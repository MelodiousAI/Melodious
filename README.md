\# Melodious



Melodious is an optical music recognition project that detects music symbols from sheet music images and converts them into graph-based representations for further processing and export.



\## Project Structure



\- `src/` : source code

\- `data/raw/` : raw dataset files

\- `data/processed/` : processed outputs

\- `outputs/` : generated files and debug outputs

\- `tests/` : test scripts

\- `notebooks/` : experiments and exploration
